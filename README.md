# Bryntum Calendar + Highcharts Integration

A demo application showcasing the integration of Bryntum Calendar with Highcharts for interactive data visualization. This project displays calendar events alongside dynamic charts that visualize event distribution per resource or per day.

## Features

- Interactive calendar with resource management
- Real-time chart updates using Highcharts
- Event visualization by resource or by day
- Resource filtering with live chart synchronization

## Installation

Install dependencies:

```bash
npm install
```

Install Highcharts:

```bash
npm install highcharts
```

## License & Trial Version

This project uses **Bryntum Calendar**, which requires a commercial license for production use.

For the trial version:

```bash
npm install @bryntum/calendar@npm:@bryntum/calendar-trial
```

For licensed users:

```bash
npm install @bryntum/calendar
```

## Running the Project

Start the development server:

```bash
npm run dev
```

Then open your browser to the provided localhost URL.
